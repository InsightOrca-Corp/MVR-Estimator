# MVR-Estimator

**Date:** 8 November 2025

The Minimum Viable Revenue (MVR) Estimator is a Streamlit-hosted web application to estimate the minimum annual revenue a company must generate to maintain financial viability.

Use the app here: https://mvr-estimator.streamlit.app/

**Important Notes**
- This app uses Python version `3.12.0`. Specific dependency versions are listed in `requirements.txt`.
